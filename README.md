# Brain games
Package contain some brain games.
## Install
```sh
curl -sSL https://install.python-poetry.org | python3 -
poetry install
make build
make package-install
```

### Run games
```sh
brain-games
brain-even 
brain-calc 
brain-gcd 
brain-progression 
brain-prime
```
